== Description ==

This plugin includes the LightWindow v2.0 javascript written by <a href="http://www.stickmanlabs.com/lightwindow/">stickmanlabs.com</a>. See <a href="http://www.stickmanlabs.com/lightwindow/">stickmanlabs.com</a> for demo and examples.

== Installation ==

1. Unzip the `lightwindow.zip` file.
2. Upload "lightwindow" folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add a class="lightwindow" attribute to any link tag to activate the lightwindow.
5. Use params="" for additional function, se link above

== Frequently Asked Questions ==

= Internal linking =

params="lightwindow_type=external"

= LightWindow width and height =

params="lightwindow_height=600,lightwindow_width=600"
